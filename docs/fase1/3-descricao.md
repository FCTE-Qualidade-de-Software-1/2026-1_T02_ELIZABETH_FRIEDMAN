## Sobre o Projeto

A **Aplicação de Gestão de Inventário Otimizada (AGIO)** é um sistema web *Open Source* desenvolvido por estudantes da Universidade de Brasília (UnB) para a disciplina de **Métodos de Desenvolvimento de Software**.

O principal objetivo do sistema é oferecer uma solução prática, segura e eficiente para o gerenciamento de inventários em empresas de pequeno e médio porte. A plataforma permite que usuários controlem os itens armazenados de forma intuitiva, possibilitando o cadastro, edição, remoção e monitoramento de componentes conforme a necessidade da organização.

Além disso, o AGIO busca simplificar o processo de administração de estoque por meio de uma interface acessível e funcionalidades que auxiliam no controle e na organização das informações do inventário.

## Principais Funcionalidades

- Criação e gerenciamento de **superusuários**;
- Sistema de **login e logout** com controle de acesso às páginas;
- Armazenamento das informações de usuários e inventário em **banco de dados**;
- Visualização completa dos itens cadastrados no inventário;
- Adição, edição e remoção de itens do inventário;
- Organização e visualização personalizada da sequência dos itens;
- Exportação dos dados do inventário para arquivos no formato **`.csv`**.

## Tecnologias e Objetivos

O AGIO foi desenvolvido com foco em:

- Facilidade de uso;
- Segurança no gerenciamento das informações;
- Organização eficiente do inventário;
- Acessibilidade para pequenas e médias corporações;
- Desenvolvimento colaborativo em ambiente acadêmico.
